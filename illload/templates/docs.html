ill.fi

all api endpoints return a json with an error (that is either true or false) and a message

/api/upload

upload takes two arguments

  • key - your upload key (can be found on your profile page)
  • files - a list of all the uploaded files, with the original file's filename being the key

upload returns the following parameters

  • error - if there was an error with your upload
  • deletion_key - the deletion key for this file
  • filename - the filename. append this to https://ill.fi/

/api/upload/delete

delete takes two arguments

  • deletion_key - the deletion key that was provided when the file was uploaded
  • file - the file you want to delete

delete returns the following parameters

  • error - if there was an error with your deletion
  • filename - the filename of the deleted file

/api/upload/delete_all

delete takes two arguments

  • key - the upload key that is linked to your account
  • email - the email you use to login

delete_all returns the following parameters

  • error - if there was an error with your deletion
  • message - either an error or success message

/api/user/exists

exists takes one argument

  • email - the email

email returns the following parameters

  • error - if there was an error with your request
  • exists - whether or not the user exists
